The excutable file is named as program 3.
The "set aperture" will allow you choose different angle. The default angle is 45 degrees.
The "rotation" will allow you rotate the cubes by clicking and holding mouse left and drag.
The "show perspective" will show the extend line of edges.
The "Hide perspective" will hide the extend line of edges.
In default, the extend line is hiden.
The "reset" will reset all manipulations.
The "exit" will exit the program.

Using the Shift key with mouse-left can do the translation.
Using the ctrl-key in conjunction with mouse-left can zoom in and zoom out.
IMPORTANT: when doing the translation and the zoom, please hold the shift key or ctrl key first,
then use the mouse-left to drag.
